# Welcome to Diaspora Node.js

Welcome to your new Diaspora blog powered by Node.js! This is a sample post to get you started.

## What is Diaspora?

Diaspora is a beautiful, minimalist blog theme that emphasizes simplicity and elegance. This Node.js version brings the classic Diaspora design to a modern Node.js backend.

## Features

- 📝 Markdown-based content
- 🏷️ Tags and Categories
- 🔍 Full-text search
- 📱 Responsive design
- 🎨 Beautiful UI
- ⚡ Fast performance
- 🔧 Easy to customize

## Getting Started

1. Add your markdown posts to `data/posts/`
2. Configure your site in `config/config.yml`
3. Run `npm start` to start the server
4. Visit `http://localhost:3000`

## Markdown Support

You can use all standard markdown features:

- **Bold text**
- *Italic text*
- `Code snippets`
- [Links](https://example.com)
- Lists and more!

Happy blogging! 🎉
